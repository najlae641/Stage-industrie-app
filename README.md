#  Gestion de Stock - ETASCOM AUTOMOTIVE

Application Web de gestion de stock et de produits, développée dans le cadre d'un stage technique au sein de l'entreprise **ETASCOM AUTOMOTIVE**.

##  Corrections Effectuées
Conformément au plan de stage, les optimisations suivantes ont été réalisées pour assurer la robustesse et la clarté du projet :

-  **Correction des types** : Conversion des champs `qte` (quantité) et `prix` en type `number` pour des calculs précis.
-  **Suppression des 'any'** : Migration vers TypeScript strict avec l'utilisation d'interfaces pour un typage propre.
- **Organisation du code** : Restructuration des dossiers et renommage des composants (ex: `sidebar`) pour une architecture professionnelle.
-  **Inputs contrôlés** : Liaison de tous les champs via `value` et `onChange` (Controlled Components).
- **Metadata** : Personnalisation du titre et de la description pour refléter l'identité de l'entreprise.

##  Technologies Utilisées
* **Framework** : [Next.js 14](https://nextjs.org/)
* **Langage** : [TypeScript](https://www.typescriptlang.org/)
* **Styling** : [Tailwind CSS](https://tailwindcss.com/)
